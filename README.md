# Backend
The backend of FlagClicked (Nuxtified).

## Run
You will need [Node.js](https://nodejs.org) and [npm](https://npmjs.org) or yarn.

NPM:

```bash

git clone https://github.com/FlagClicked/Backend

cd Backend

npm install

# run with live reload

npm run dev

# start server for production

npm start
```

yarn:

```bash

git clone https://github.com/FlagClicked/Backend

cd Backend

yarn install

# run with live reload

yarn dev

# start server for production

yarn start
```
